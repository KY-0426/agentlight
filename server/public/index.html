Agent Light 是面向 AI coding agent 的桌面状态伴侣：透明桌宠、ESP32 RGB 灯、CLI/API 集成，以及运行在同一云托管域名上的账号与 Token 排行榜。

为什么需要 Agent Light

桌宠始终可见

置顶透明窗口，可拖到桌面任意位置。蓝 / 黄 / 绿 / 红四种状态，对应待命、工作中、已完成、需关注。

脚本与 CLI 集成

本地 POST /api/state 与 agent-light state completed "Done"，让自动化流程也能更新状态。

可选实体 RGB

ESP32 USB 串口驱动同一套状态语义；硬件失败不会拖垮桌面体验。

四种标准状态

standby 蓝色 · 等待任务
working 黄色 · 正在执行
completed 绿色 · 完成待确认
attention 红色 · 需要人工介入

微信云托管 · 同一域名

本页面与 API 运行在同一容器内。桌面端「设置 → 账号 → 服务端地址」填写下方域名即可登录、同步 Token 排行榜。

健康检查：/health · API 前缀：/api/*
